angularify.semantic.popup
===============================

`angularify.semantic.popup` - popup directive for angular.js.

Usage
-------------------------------

```html
<i class="heart circular icon hover" 
   popup="{title : 'Hello this is content', 
           content : 'Popup content', 
           position: 'bottom', 
           size: 'small'}"></i>
```

`position` - can be:

  * bottom
  * top
  * right
  * left

`size` - can be:

  * small
  * large
